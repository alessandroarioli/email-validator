How it works: We perform a real SMTP handshake with the domain's mail server — EHLO → MAIL FROM → RCPT TO — and read the server's response code. A 250 means the mailbox was accepted; a 550 means it was rejected. Some servers (including Gmail) use catch-all policies or greylisting, which may result in an inconclusive result.
